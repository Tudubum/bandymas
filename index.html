Terminalo Komandos

  • ls / dir(window'sams) - parodo aplankalus ir failus dabartineje direktorijoje. (dar kazkas su ls -a)
  • cd - changeDirectory
  • TAB - skirtas uzpildyti terminalo komanda arba pateikti pasiulymus
  • CTRL C - atsaukia vykdoma veiksma / nutraukia komanda
  • vidurinis/desinys peles mygtukas - iklijuoja teksta i komandine eilute
  • cd .. - zingsnis atgal
  • ARROW UP/DOWN - vaiksciojimas per terminalo komandu istorijas

Git Komandos

  • git init - inicializuoja git repozitorija
  • git status - parodo git repozitorijos statusa
  • git add - prideti failus prie paruostu commit'inimui failu
  • git commit -m "zinute" - sukuriama kodo versija su zinute
